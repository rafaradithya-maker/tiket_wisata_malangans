## 📊 Troubleshooting Guide

### ❌ Problem: "Connection refused"
**Solution:**
1. Buka XAMPP Control Panel
2. Klik "Start" untuk Apache & MySQL
3. Tunggu sampai berwarna hijau
4. Refresh browser

### ❌ Problem: "Database not found"
**Solution:**
1. Buka phpMyAdmin: `http://localhost/phpmyadmin`
2. Klik tab "Import"
3. Browse file: `db_init.sql`
4. Klik "Import"
5. Tunggu sampai success

### ❌ Problem: "Session not working / Logout sendiri"
**Solution:**
1. Buka `http://localhost/wisata_malang/status_check.php`
2. Cek "Session Support" harus ✓ OK
3. Clear browser cookies:
   - Tekan F12
   - Tab Application
   - Hapus cookies localhost
4. Refresh dan coba login lagi

### ❌ Problem: "File not found / 404"
**Solution:**
1. Pastikan folder: `C:\xampp\htdocs\wisata_malang\`
2. Pastikan file ada di folder
3. Cek URL benar: `http://localhost/wisata_malang/file.php`
4. Buka diagnostic: `http://localhost/wisata_malang/diagnose.php`

### ❌ Problem: "API returns error"
**Solution:**
1. Buka browser DevTools (F12)
2. Tab "Network"
3. Coba akses endpoint
4. Lihat response
5. Periksa error message
6. Buka `test_api.php` untuk test terstruktur
