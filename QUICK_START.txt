🛠️ TROUBLESHOOTING
===================

❌ Aplikasi tidak bisa dibuka?
   • Pastikan XAMPP Apache running (berwarna hijau)
   • Pastikan MySQL running (berwarna hijau)
   • Akses http://localhost/phpmyadmin untuk test koneksi

❌ Login tidak bekerja?
   • Buka /status_check.php
   • Cek apakah "Session Support" status OK
   • Clear browser cookies dan coba lagi

❌ Database error?
   • Buka http://localhost/phpmyadmin
   • Import file: C:\xampp\htdocs\wisata_malang\db_init.sql
   • Pastikan database "tiket_malang" ada

❌ File not found / halaman 404?
   • Pastikan struktur folder di C:\xampp\htdocs\wisata_malang\
   • Refresh browser
   • Cek console browser (F12) untuk error detail
